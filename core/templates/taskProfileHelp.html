Task execution profile:

The plot shows the progress of a task in time.

You can use multi-select inputs on the top to select particular types and statuses of jobs.

Also you can select a task progress unit: job, events, or files. It will be applied to y-axis. Since active jobs have not processed input events or files, it is shown as 0 until they finished.

The plot is interactive, if one hover a point the extra information as pandaid of a job will be shown in tooltip. If you can click on a point a new tab for the selected job will be opened. You can click on legend items to hide points associated to its category. Vertical lines show start (blue) and end (black) of task attempts.
