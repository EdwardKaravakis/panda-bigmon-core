Dataset information:

This page accumulates all the information of JEDI dataset.

See the JEDI twiki for documentation on PanDA JEDI dataset status transitions and schema.

Be sure to familiarize yourself with JEDI based analysis differences from a user perspective, important ones relate to datasets.
