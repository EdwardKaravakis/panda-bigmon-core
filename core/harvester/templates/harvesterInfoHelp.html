Harvester list help:

Available filters for this page:

  • Information for a specific computingsite. E.g. /harvesters/?instance=act403&computingsite=ANALY_UIO

  • Information for a specific resourcetype. E.g. /harvesters/?instance=act403&resourcetype=SCORE

  • Information for a specific computingelement. E.g. /harvesters/?instance=act403&computingelement=None

  • Information for a specific status. E.g. /harvesters/?instance=act403&status=running

  • Information for a specific workerid or workerids. E.g. /harvesters/?instance=act403&workerid=759585,759584

  • Information for the last N days. E.g. /harvesters/?instance=act403&days=3

  • Information for the last N hours (default 24 hours). E.g. /harvesters/?instance=act403&hours=48

If you need information about workers which are associated with a specific computingsite, use computingsite parameter instead of instance parameter. E.g. /harvesters/?computingsite=ANALY_UIO

See the Harvester documentation on workers status transitions.
