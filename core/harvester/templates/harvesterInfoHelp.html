Harvester list help:

This page is a summary of Harvester workers. It is better to specify instance as this is the indexed field and therefore works faster. The default time range is last 12 hours, you can change it using hours=N filter.

See the Harvester documentation on workers status transitions.

Available filters for this page:

  • by instance, e.g. /harvesters/?instance=CERN_central_A
  • by computingsite, e.g. /harvesters/?instance=CERN_central_A&computingsite= MWT2
  • by resourcetype, e.g. /harvesters/?instance=CERN_central_A&resourcetype=SCORE
  • by computingelement, e.g. /harvesters/?instance=CERN_central_A&computingelement=iut2-gk02.mwt2.org:9619
  • by status, e.g. /harvesters/?instance=CERN_central_A&status=running
  • by workerid or list of workerids, e.g. /harvesters/?instance=CERN_central_A&workerid=467254600,467374162
