Running production tasks page:

This page summarizes the status of different productions such as MC, derivation, reprocessing and other.

Only active tasks are taken into account and shown by default. The plots on the top represents the current status of a production in number of events and allocated slots.

Use the row of drop-down menus to more narrow selection of tasks by processing type, working group, campaign etc.

Click on icon next to Events title of number of events pie charts section to see extra charts.

Click on show historical trend to see a line plot showing how number of waiting and running events have been changing for last 7 days.

Columns in task list table can vary depending on the chosen preset. Values in bold are clickable filters for the page. icons are links to corresponding pages in ProdSys2 UI.

If you wish to include ended tasks to selection, add &show_ended_tasks=true to URL together with &days=N where N is number of days. Please use it with other filters (hashtags, reqid etc) to decrease number of selected tasks and therefore the load on DB.
