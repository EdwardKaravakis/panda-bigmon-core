Running production tasks page:

This page summarizes the status of running productions such as MC, derivation, reprocessing and other. Only active tasks are taken into account and shown by default. If you wish to include ended tasks to selection, add one of supported time range options to URL:

  • &days=N for last N days
  • &hours=N for last N hours
  • &date_from=YYYY-MM-DD and &date_to=YYYY-MM-DD for custom time range
Please use it with other filters (hashtags, reqid, campaign etc) to decrease number of selected tasks and therefore the load on DB.

The plots on the top represents the current status of a running production in number of events and allocated slots.

Use the row of drop-down menus to more narrow selection of tasks by processing type, working group, campaign etc.

Click on icon next to Events title of number of events pie charts section to see extra charts.

Click on show historical trend to see a line plot showing how number of waiting and running events have been changing for last 7 days.

Columns in task list table can vary depending on the chosen preset. Values in bold are clickable filters for the page. icons are links to corresponding pages in ProdSys2 UI.
